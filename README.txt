#Penggunaan Program (Bahasa Indonesia)
1. Pastikan server MySql anda menyala (Localhost)
2. Insialisasi basis data dengan menjalankan SQL yang terdapat
di dalam berkas 'PembayaranSPP.sql'
3. Buka Netbeans (current version 8.2), kemudian buka projectnya
4. Masukan library MySql Connector ke Project dengan cara:
Klik Kanan Pada 'Libraries' -> Add Jar/Folder -> Pilih berkas
'mysql-connector-java-5.1.47-bin' -> Klik Open
5. Program langsung bisa dijalankan

#How To Use This Program (English)
1. Make sure your MySql server is on (Localhost)
2. Initialize the database by running the existing SQL
in the 'PembayaranSPP.sql' file
3. Open Netbeans (current version 8.2), then open the project
4. Enter the MySql Connector library into the Project by:
Right-click on 'Libraries' -> Add Jar / Folder -> Select file
'mysql-connector-java-5.1.47-bin' -> Click Open
5. Direct programs can be run
